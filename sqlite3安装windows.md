## 安装sqlite3

- 将sqlite3.exe 放到 C:/windows/system32 

- 打开cmd，敲入命令 sqlite3

- 出现如下 ，就成功了

  ```
   C:\Users\Goomy>

  C:\Users\Goomy>

  C:\Users\Goomy>sqlite3

  SQLite version 3.7.14.1 2012-10-04 19:37:12

  Enter ".help" for instructions

  Enter SQL statements terminated with a ";"

  sqlite>
  ```
